# SilentFailCheck

This gem aims to track and log silent failure.

## Installation

Add this line to your application's Gemfile:

    gem 'silent_fail_check'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install silent_fail_check

## Usage

Don't use it in production.
